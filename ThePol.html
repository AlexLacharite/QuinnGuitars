The Pol


A hommage to tradition, this Les Paul style model is based on a timeless classic, with some improvements.
This solid body electric guitar features a set-thru neck construction, assembled with the utmost care and precision.
The 12.5° angled headstock is reinforced with front and rear headplates and features a handcarved thumb stop, for added strength and an easy reference point for the fretting hand.
This model comes standard with a bound fretboard, 2-pickup configuration, Schaller STM roller bridge and stop tailpiece.
Custom nut width, neck shape and electronics available.
